How to write a Security Plan (that everyone will read and follow) Part. 1

Date Sun 07 September 2014
  • Security Planning
  • Security Plan

An essential element of field security is a written Security Plan that is read and followed by everyone it applies to. Writing a good Security Plan is no different to other project documents, such as a donor proposal.

A good Security Plan should be:

  • Concise
  • Relevant
  • Accessible
